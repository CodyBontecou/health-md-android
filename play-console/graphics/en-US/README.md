# Graphics Assets

Add the following files to this directory:

## Required
- **featureGraphic.png** (1024x500px) - Displayed on store listing page
- **icon.png** (512x512px) - App icon (high-res version)

## Optional
- **promoGraphic.png** (180x120px) - Used for promotional materials
- **tvBanner.png** (1280x720px) - For Android TV listings

## Tips
- Use PNG format with transparency where appropriate
- Ensure consistent branding across all assets
- Feature graphic should showcase your app's main value
